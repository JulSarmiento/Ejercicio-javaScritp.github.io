## Método de evaluación

Se creará un proyecto en git con la misma estructura presentada en este repositorio
y se iniciará a resolver los puntos cada uno en una carpeta individual por ejercicio.

Ejemplo

```terminal
. mis-ejercicios-js
├── 1. Basico
│   ├── 1.1 ejemplo.js
│   ├── 1.2 example.js
│   ├── 1.. ejemplo.js
├── 2. Intermedio
│   ├── 2.1 ejemplo.js
│   ├── 2.2 example.js
│   ├── 2.. ejemplo.js
├── 3. Avanzado
│   ├── 3.1 ejemplo.js
│   ├── 3.2 ejemplo.js
│   ├── 3.. ejemplo.js
```
